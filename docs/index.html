Strawberry is a new flexbox based CSS micro-framework

Installation

$ npm i --save strawberry-css

or

<link rel="stylesheet" href="https://unpkg.com/strawberry-css/dist/strawberry.min.css">

Features

  • It's light: less than 18Kb
  • Itâ€™s powerful: create Flexbox-based layouts in few lines of code
  • No preset graphic style: it helps you create layouts without messing up your life
  • No conflicts with other used frameworks in the same project: all classes start with the sb- prefix
  • Easy to override: strawberry never uses !important
